- Score > 90: Badge "Thưởng ca (+50k - 100k)"
- Score < 70: Badge "Đào tạo lại + Cảnh cáo"
- 3x < 70 in one month: Critical alert "Xem xét cho nghỉ"
- Manual financial adjustments: Nhẹ (-20k-50k), Trung bình (-50k-100k), Nặng (-200k + Nghỉ ca)
- Manual rewards: Khách tip riêng, Khách gọi tên khen
- Top performers (avg > 90): Target reward +200k-500k
